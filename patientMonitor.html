About project


This is a patient monitoring app for nurse practitioners.

- React (Modern JS Framework) Application
- DB (MongoDB) is connected to store information
- ML Algorithm is used for prediction of hepatitis

Features:
- Sign in/Sign up
- Check patient information
- Add daily information
- Add/View daily motivational tips
- Add/View emergency alert
- Use medical checklist (AI)
