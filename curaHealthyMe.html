About project


CuraHealthyMe is a web application that provides an easier solution for patients to easily access the clinical services.
It gives easier collaboration between doctors and patients with ease of access to clinical services with less telephone calls for the appointments.

- Spring Boot MVC Application consumes REST APIs
- Used DB (MySQL) to store information

Features:
- Sign in/Sign up
- Find Family doctor
- Book/Reschedule/Cancel/View appointment
- View medical documents
- About us, Contact us, News Etc.
